# vantage-fx-code-challenge

## Built With

* React
* Typescript
* styled-component
* material-ui
* service-worker

## Quick Start

* Install Library

    ``` js
    npm install
    ```

* Run developer mode

    ``` js
    npm run dev
    ```

* Run production mode

     ``` js
    npm run build
    ```

    ``` js
    npm run start
    ```

    The server will run at `http://localhost:8888`
